# Overview

This repository contains the solution for the laboratory assignment **"Lab 2"** on automating the development of console applications. The project is based on the template provided for FPMI students.

# Usage

This project serves as a practical exercise for working with Git, GitHub, and the Replit online development environment. The goal is to learn how to import a repository, create branches, and edit project documentation.

**Student:** Neyfeld Kirill, Group 11

# Building

This project is designed to be opened and run in the Replit environment ([https://repl.it](https://repl.it)). To work with the project, import this repository into Replit. The application is a console program written in C.
